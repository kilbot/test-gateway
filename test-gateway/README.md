## WooCommerce POS Test Gateway Gateway

Test description

### Instructions

1. Check [releases](https://github.com/kilbot/test-gateway/releases) for the latest version of the plugin.
2. Download the **woocommerce-pos-test-gateway-gateway.zip** file.
3. Install & activate the plugin via `WP Admin > Plugins > Add New > Upload Plugin`.
4. Go to `WP Admin > WooCommerce POS > Settings > Checkout > enable` the WooCommerce POS Test Gateway Gateway.

### Development

To customize this gateway, edit the `wcpos-test-gateway.php` file and implement your payment processing logic in the `process_payment()` method.
